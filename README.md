# AI Agent Skills & Contexts

Welcome to the personal AI Agent repository of **Konstantin Sharlaimov ([github.com/digitalentity](https://github.com/digitalentity))**.

This repository is a centralized hub for **Skills** and **Context files** used to extend, configure, and align the behavior of agentic AI coding assistants (such as Antigravity, Gemini, and Claude Code) across various development workspaces.

## 🚀 Overview

AI coding assistants are most effective when provided with grounded guidelines, project standards, and codebase architecture. This repository serves two purposes:
1. **Skills (Modular & Centralized)**: Packaged executable instructions, scripts, and resources that can be dynamically discovered and executed by AI agents in specific environments.
2. **Context Files (Project-Specific)**: Standardized configuration and guide templates (e.g., `CLAUDE.md`, `GEMINI.md`) that reside in target project repositories to establish rules of engagement.

## ✍️ Skills

Skills are modular, executable instructions, templates, and scripts designed to guide AI agents on specific tasks.

## 📄 Context Files

Unlike modular, executable **Skills** stored centrally in this repository, **Context Files** are project-specific guidance files placed directly at the root of target repositories to serve as the "source of truth" for the AI assistant. These files prevent general-purpose AI hallucinations and "vibe coding."
